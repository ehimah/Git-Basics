This is an introductory file for this repository

git init - creates a new git repository in the current directory

git status - to view the status of the git repository. THis display current branch, and if there are any new / untracked files

git commit - this records / stores the current tracked changes into the repository

git add - this adds the content of the specified file to the git repository, and makes it ready to be committed

git log - this is show the commits history of a repository

git branch - this command is used to create a new git branch on the repository

git checkout - this command is used to switch to an existing branch

git merge - this command is used to merge changes from one branch into the current branch
